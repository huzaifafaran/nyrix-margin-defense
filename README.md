# Nyrix AI Margin Defense System (Pilot)

This is a Streamlit prototype built for Lucky Cement to demonstrate AI-driven financial and operational intelligence.

## Features
- **Margin Radar:** P&L data visualization.
- **Scenario Simulator:** Interactive "What-If" analysis for costs.
- **COP Deep Dive:** Raw material and power mix optimization.
- **Executive Chatbot:** RAG-based query interface (simulated).

## Deployment
Deployed on Streamlit Cloud.
